# Hotel Management System (HMS)

**HMS** — це сучасний Fullstack веб-застосунок для повної автоматизації роботи готелю.

## 🏨 Про проєкт

Проєкт призначений для цифровізації взаємодії між гостями та адміністрацією готелю. Система дозволяє гостям зручно підбирати номери та здійснювати оплату онлайн, а персоналу — оперативно керувати заїздами, технічним станом фонду та фінансовою звітністю.

## 🚀 Технологічний стек

- **Backend:** Node.js, Express, TypeScript.
- **Frontend:** React (Vite), TypeScript, Tailwind CSS.
- **Database:** PostgreSQL.
- **ORM:** Prisma.
- **Автентифікація:** JWT (Access + Refresh Tokens) + Bcrypt (хешування паролів).
- **Інтеграції:** LiqPay API (платежі та повернення), Nodemailer/SMTP (відновлення пароля).
- **Тестування:** Jest (Unit/Integration), Playwright (E2E), Stryker (Mutation).

## 🛠 Ключовий функціонал

### Для клієнтів (Гостей)

- **Інтерактивний каталог:** Пошук за місткістю, зручностями та типами ліжок.
- **Розумне бронювання:** Вибір конкретних дат через календар із логікою "split-days" (заїзд/виїзд в один день).
- **Оплата:** Реальна інтеграція з LiqPay для оплати карткою або вибір оплати при заселенні.
- **Кабінет гостя:** Перегляд статусів поїздок, скасування броні та залишення верифікованих відгуків.
- **Безпека:** Відновлення пароля через електронну пошту.

### Для персоналу (Ресепшіоністів та Адмінів)

- **Dashboard:** Таблиця всіх бронювань готелю з пошуком та фільтрацією.
- **Управління заїздами:** Процедури Check-in та Check-out, підтвердження оплати готівкою.
- **Direct Booking:** Створення бронювання для клієнта "з вулиці" з автоматичною реєстрацією акаунта.
- **Журнал робіт:** Ведення логів технічного обслуговування з автоматичним блокуванням номерів у системі продажів.

### Тільки для Адміністраторів

- **Керування інвентарем:** Повний CRUD для номерів, категорій, зручностей та типів ліжок.
- **Asset Manager:** Вбудований менеджер зображень із завантаженням файлів на сервер та їх видаленням.
- **Фінанси:** Оформлення повернення коштів (Refund) через LiqPay API.
- **Персонал:** Створення та видалення акаунтів ресепшіоністів.
